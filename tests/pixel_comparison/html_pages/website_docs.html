Installation

Get up and running with DocHub in under 5 minutes. This guide covers installation via npm, yarn, and from source.

Prerequisites

RequirementMinimum VersionRecommended
Node.js18.020.x LTS
npm9.010.x
OSmacOS, Linux, WindowsmacOS / Linux

Install via npm

The fastest way to get started is installing the CLI globally:

# Install globally
npm install -g @dochub/cli

# Create a new project
dochub init my-project
cd my-project
ⓘ
Note: If you prefer not to install globally, you can use npx @dochub/cli init instead.

Configuration

After installation, create a dochub.config.js file in your project root:

export default {
  title: "My Documentation",
  theme: "default",
  sidebar: true,
  search: { provider: "local" }
};
⚠
Warning: The theme field is required. Omitting it will cause a build error.
✔
Tip: Run dochub dev to start a local development server with hot reload at localhost:3000.
